# Try Out Kompetensi Teknis DJPb

Aplikasi Streamlit untuk latihan try out kompetensi teknis DJPb.

## Jalankan lokal

```bash
python -m streamlit run app.py
```

Password login: `PPA2KALBAR`

## Catatan v34

- TO umum: 100 soal, 150 menit.
- TO per materi: 50 soal.
- Sumber soal: All, Bank Soal 1, Bank Soal 2.
- Jika TO per materi memilih Bank Soal 1 tetapi soal yang dapat tampil unik kurang dari 50, aplikasi otomatis menambah kekurangan dari Bank Soal 2 pada materi yang sama.
- Tombol Beranda tersedia di sidebar dan halaman hasil.
